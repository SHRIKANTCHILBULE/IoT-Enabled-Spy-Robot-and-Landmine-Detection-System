# IoT-Enabled-Spy-Robot-and-Landmine-Detection-System
Final Year B.E. Automation and Robotics Project on an IoT-enabled Spy Robot featuring live video streaming, landmine detection, GPS tracking, biometric authentication, fire and gas detection, and remote monitoring using ESP8266.
# Design and Analysis of Spy Robot with Advanced Surveillance

## Project Overview
This project presents an IoT-enabled Spy Robot designed for surveillance and operation in hazardous environments. The system integrates multiple sensors and communication technologies to provide real-time monitoring, remote control, and threat detection.

## Features
- Live video streaming
- Remote control using Wi-Fi
- GPS location tracking
- Landmine detection using metal sensors
- Fire detection
- Toxic gas detection
- Fingerprint-based authentication
- Solar-assisted charging system
- Real-time monitoring and alerts

## Technologies Used
### Hardware
- NodeMCU ESP8266
- Metal Detector Sensor
- GPS Module (Neo-6M)
- Fire Sensor
- Gas Sensor
- Fingerprint Sensor (R307)
- L298N Motor Driver
- DC Motors
- Camera Module
- Solar Panel
- 12V Battery

### Software
- Arduino IDE
- Embedded C/C++
- IoT Communication
- Wi-Fi Networking

## Project Domain
Internet of Things (IoT), Automation and Robotics, Embedded Systems, Surveillance Systems.
